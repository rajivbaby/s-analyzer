This is a simple program to plot S-parameter files in C#
Formats: Magnitude in dB and Smith Chart
To run exe, download files: S-Analyzer.deps.json, S-Analyzer.dll, S-Analyzer.pdb, S-Analyzer.runtimeconfig.json with the S-Analyzer.exe and keep in the same folder as the exe file.
It is free to use and modify for your personal use.
